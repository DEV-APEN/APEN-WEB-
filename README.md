## 🏗️ Plataforma Corporativa APEN
Plataforma web de ingeniería y administración de proyectos para el sector energético. Diseñada bajo una estética de **"Gabinete Técnico"**, la web proyecta precisión institucional, rigor normativo y capacidad de integración 360°.

### 🚀 Pilares del Proyecto
1. **Ecosistema 360°**: Visualización interactiva de la integración de servicios (Obra Civil, Administración, Consultoría y Servicios Técnicos).
2. **Ruta de Certeza**: Diagrama de flujo tecnológico animado que valida las certificaciones ISO (9001, 14001, 45001) y registros oficiales (Achilles PEMEX, CFE).
3. **Identidad Institucional**: Arquitectura de información minimalista optimizada para la conversión en licitaciones y contratos técnicos.

### 🛠️ Especificaciones Técnicas
- **Frontend**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS (Optimización de escalas y densidad visual)
- **Animaciones**: Framer Motion (Flujos eléctricos y micro-interacciones)
- **Iconografía**: Lucide React

### 📦 Instalación y Despliegue
Para clonar y ejecutar el entorno de desarrollo localmente:

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```
